# **App Name**: CertiFlow Industrial

## Core Features:

- Quality Report Generation: Create new industrial quality reports ('Novo Laudo') by inputting identification details, equipment models, serial numbers, client information, and operation type (e.g., recovery or sale).
- Tabbed Quality Control Modules: Navigate through different aspects of a quality report, including 'Identificação', 'Testes', 'Serviços', 'Qualidade', 'Mídia', and 'Garantia', using an intuitive tabbed interface for structured data entry and review.
- Secure Record Management: Save new and update existing industrial quality reports securely in a database, allowing for efficient archiving ('Arquivar no Banco') and retrieval of records.
- Reporting & Data Overview: Access and view compiled quality reports, providing a structured overview and summary of industrial data to monitor and analyze quality metrics.
- AI-Assisted Data Entry: Utilizes a generative AI tool to suggest and pre-fill fields like equipment specifics or common test procedures based on partial inputs or historical data, improving data entry speed and accuracy.
- User Authentication & Authorization: Manage different user roles, such as 'Mestre Brazilian' (Master User), ensuring secure login and appropriate permissions for accessing and managing system functionalities and sensitive data.
- Media Attachment for Reports: Allow users to upload and link relevant media (e.g., images of equipment, supporting documents) directly to specific quality reports under the 'Mídia' section for comprehensive documentation.

## Style Guidelines:

- Primary color: A deep steel blue (#203F60), evoking professionalism, stability, and reliability suited for industrial control systems.
- Background color: A very light, desaturated blue-grey (#ECF1F6), providing a clean, subtle foundation that enhances readability in a light color scheme.
- Accent color: A vibrant aqua blue (#26A3BB), selected as an analogous color to the primary blue to provide dynamic contrast and highlight interactive elements, echoing the visual cues in the provided image.
- Headline and body font: 'Inter' (sans-serif), chosen for its modern, objective, and highly legible characteristics, suitable for conveying detailed information in a technical and industrial application.
- Use simple, clean, and outlined icons that provide clear visual cues for navigation and actions, maintaining a functional and modern aesthetic, as seen in the existing UI elements.
- Adopt a structured and modular layout with well-defined form sections, intuitive tabbed content navigation, and a prominent persistent top header for branding and key system actions, ensuring a clear information hierarchy.
- Implement subtle animations and transition effects, such as smooth hover states for buttons and tabs, to provide positive user feedback and enhance the fluidity of interactions within the application.
